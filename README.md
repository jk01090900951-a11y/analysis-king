# 분석왕 (Analysis King) — 최종 정리본

이 패키지는 지금까지의 대화에서 **확정·추가·수정·보완된 내용만** 모은 최종본입니다.
픽앤포인트 시절 문서, Render 배포 관련 문서, 폐기된 정책 분석 문서 등 더 이상 쓰지 않는 파일은 전부 제외했습니다.

## 폴더 구조

```
코드/              → 실제 프론트엔드/백엔드 코드 (17개 파일)
문서_정책/          → 포인트 정책, 이용약관, 이용안내, FAQ (4개)
문서_기획/          → PRD, API 명세서, 관리자 모드 명세서, 광고 관리 설계, 디자인 시안, UX 플로우 (12개)
문서_배포가이드/     → 카페24 VPS 설치, GitHub 저장소 생성, Git 설치, 코드 업로드, 환경변수 관리 (7개)
문서_참고/          → 포인트정책 상세가이드, 서비스 가이드라인 총정리, 이슈트래커 (3개)
분석왕_광고수익_시뮬레이션.xlsx  → 수익 시뮬레이션 (최상위)
```

## ⚠️ 중요 — 이 패키지만으로는 실행이 안 됩니다

여기 있는 건 **이번 대화에서 확정/수정된 파일들만**입니다. `package.json`, 빌드 설정 파일, 그리고 제가 손대지 않은 나머지 원래 프로젝트 파일들(예: `AdminMatches.tsx`, `CategoryMenu.tsx` 등)은 포함되어 있지 않습니다.

**실제 사용 방법**: 원래 갖고 계신 전체 프로젝트 폴더에서, 여기 있는 파일과 **같은 이름의 파일을 이걸로 덮어쓰기** 하시면 됩니다. (`schema.ts`, `db.ts`, `routers.ts`는 특히 이번에 크게 바뀐 핵심 파일입니다.)

## 핵심 원칙 요약 (문서_참고 폴더에 상세 내용 있음)

1. 포인트는 100% 무상 지급 (가입/추천인 포인트 폐지)
2. 배팅 구조 없음, 유저 글쓰기 없음 (AI 분석가 20명 전용 콘텐츠)
3. 조회 시에만 포인트 지급, 광고는 완전 별개(이탈 시 팝업 + 배너, 5분 쿨다운)
4. 출금은 10,000P 이상, 1,000P 단위
5. 유료 구독 없음
6. "AI"라는 단어는 사용자 화면에 노출 안 함 (관리자 화면 제외)
7. "전문가"라는 단어도 사용 안 함 (스포츠전문분석/분석가로 대체)

## 아직 확정 안 된 것 (문서_참고/이슈트래커 참고)

- 조회자 포인트 정확한 금액
- 사업자등록 여부 (교환소 실사용 시 필요)
- 야구 투수 데이터 파서 (실제 API-Sports 응답 확인 후 완성 예정)
